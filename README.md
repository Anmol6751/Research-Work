This project investigates a CNN-based facial recognition system, highlighting its architecture, performance metrics, and practical applications.## AbstractFacial recognition has become a pivotal application in computer vision, driven by advancements in deep learning and convolutional neural networks (CNNs). This study evaluates a custom CNN model designed to extract hierarchical features for accurate classification. By addressing challenges such as overfitting, dataset bias, and computational efficiency, this research offers insights for both academic and industrial applications.## Key FeaturesHierarchical Feature Extraction: Automatically extracts features from low-level edge detection to high-level facial representation.Robust Preprocessing: Includes face detection, alignment via facial landmarks, and uniform resizing to 224x224 pixels.Data Augmentation: Improves generalization through random rotation, horizontal flipping, and brightness adjustment.Standardized Evaluation: Assessed using Accuracy, Precision, Recall, F1-Score, and AUC.## Model ArchitectureThe system utilizes a sequential CNN architecture:Layer NameFunction Convolutional LayerExtracts spatial features using filtersReLU ActivationIntroduces non-linearityMax Pooling LayerDownsamples feature maps to reduce complexityFully Connected LayerIntegrates extracted features for classificationDropout LayerMitigates overfitting by random unit dropoutOptimizer: Adam Loss Function: Categorical Crossentropy ## MethodologyDataset Selection: Utilizes diverse, large-scale datasets such as VGGFace2 and CelebA to minimize bias.Preprocessing: Normalizes pixel values to a range of [0, 1] to stabilize training.Data Splitting: Divided into Training (70%), Validation (15%), and Test (15%) sets.Training: Employs transfer learning with fine-tuning (e.g., ResNet-50 or VGG16) and early stopping based on validation performance.## AuthorsAnmol Shrotriya - VIT Bhopal University Aditya Jain - VIT Bhopal University Dr. Abha Sharma - VIT Bhopal University ## Citation[!IMPORTANT]This work is currently in preparation/under review.
